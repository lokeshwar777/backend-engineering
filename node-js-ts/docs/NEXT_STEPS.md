# Flow

3. folder structure & routing - modular(MVC)
4. linting & formatting - eslint, prettier
5. docker setup - compose(dev, prod)
6. global middlewares - validation, error handling, logging

10. auth (jwt, session, OAuth) & RBAC - hashing, tokens, cookies
11. testing - unit, integration, e2e, CI
12. performance - caching, rate-limiting, scaling, optimisation
13. api docs - postman, swagger
14. security - CORS

15. api design & docs - OpenAPI/Swagger draft, request/response schemas
16. api docs UI & client artifacts - Swagger UI / Redoc, generated client SDKs, Postman collections
17. security hardening - helmet, CORS, input sanitization, secrets rotation, dependency scanning
18. migrations & seeding - migration tool, seed scripts, versioned schema changes
19. caching infra - Redis setup, cache invalidation strategy, TTL policies
20. observability & monitoring - structured logs, traces, metrics, health checks, alerting
21. ci/cd & deployment - pipeline, environments (dev/staging/prod), image registry, blue/green or canary flows
22. secrets & config management - env segregation, secret store (Vault/Cloud KMS), runtime config
23. backups & disaster recovery - DB backups, restore plans, retention policies
24. data protection & compliance - encryption at rest/in transit, PII handling, audit logs
25. api governance & versioning - semantic versioning, breaking-change policy, deprecation plan
26. feature flags & rollout - feature flag system, gradual rollouts, kill-switch
27. observability-driven optimisation - profiling, flamegraphs, slow-query analysis, capacity planning
28. maintenance & runbook - runbooks, on-call playbooks, incident postmortems
